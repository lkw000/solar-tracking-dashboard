Data Input

Upload recorded system data or connect the Arduino to update the dashboard with live measurements.

CSV / live format:

time,voltage,current,power,battery,tracking

Main formula:
Power (W) = Voltage (V) × Current (A)

For live Arduino data, print each reading in this order: time, voltage, current, power, battery, tracking. Example: 10:00,18.2,1.5,27.3,72,ON.
